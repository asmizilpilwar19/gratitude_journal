# Gratitude Garden

Gratitude Garden is a peaceful desktop application built with **Python** and **Tkinter** that encourages users to develop a daily gratitude habit. Every gratitude entry plants a new seed in a virtual garden, allowing users to visually watch their garden grow over time as they continue practicing mindfulness.

Designed with a calming green theme and a simple, user-friendly interface, the application makes daily journaling feel rewarding by turning positive reflections into a beautiful collection of plants and flowers.

---

## Overview

Practicing gratitude has been shown to improve mental well-being, reduce stress, and promote a positive mindset. Instead of storing journal entries in a plain text list, Gratitude Garden transforms each daily reflection into a virtual plant, creating a visual representation of personal growth and consistency.

Users can write one gratitude entry each day, which is automatically saved locally. Each new day adds another plant to the garden, making mindfulness both enjoyable and motivating.

---

## Features

### Daily Gratitude Journal

- Write one gratitude entry per day
- Prevents duplicate entries for the same day
- Automatically saves all journal entries
- Simple and distraction-free writing experience

### Virtual Garden

- Every gratitude entry plants a new seed
- Garden grows over time with various plant emojis
- Visual representation of consistency
- Displays total number of planted seeds

### User Interface

- Cute pastel green theme
- Clean and beginner-friendly interface
- Custom animated buttons with hover effects
- Responsive garden canvas
- Easy-to-use layout

### Data Storage

- Stores entries locally in JSON format
- Automatically loads previous entries when the application starts
- No internet connection required

## How It Works

1. Open the application.
2. Write something you're grateful for.
3. Click **Plant Seed**.
4. Your gratitude entry is saved automatically.
5. A new plant appears in your virtual garden.
6. Return each day to continue growing your garden.

The application allows one gratitude entry per day, encouraging users to build a consistent mindfulness routine.
